kn., 2024

Interactive knot light and sound object
LED rope light, modified bells' parts, motors, acrylic box, circuit board, color and light sensor, electronics
aprx. 10' x 10' x 10'
Based on Trefoil knot variation, kn. is an interactive object that responds to color and produces light and sound combinations. The input is a color and light sensor and the output is based on the color the sensor detects. The object is programmed to recognize 8 colors. Each color corresponds to a sound note. 8 colors (rainbow + white) = 8 notes (octave + C). When a color is recognized, kn. plays a set of 4 notes starting and ending with the note corresponding to the color and then displays the color. At startup kn. cycles thru all the possible color and sound combinations in a scale/ rainbow order.
kn. is a sound and light sculpture that doubles as a lamp. Just pick a color!
