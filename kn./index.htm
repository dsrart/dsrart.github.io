kn., 2024 (prototype)

Interactive knot light and sound object
LED rope light, modified bells, motors, acrylic box, circuit board, RGB color sensor, electronics
aprx. 10" x 10" x 10"
kn. is the scientific abbreviation of a knot. Based on Trefoil knot variation, kn. is an interactive object that responds to color and produces light and sound combinations. The input is a color sensor and the output is based on the color the sensor detects. The object is programmed to recognize 8 colors. Each color corresponds to a sound note. 8 colors (rainbow + white) = 8 notes (octave + C). When a color is recognized, kn. plays a set of 4 notes starting and ending with the note corresponding to the color and then displays the color. At startup kn. cycles thru all the possible color and sound combinations in a musical scale/ rainbow color order.
kn. is a sound and light sculpture that doubles as a lamp. Just pick a color!


The making of kn.(prototype)
Acrylic box was customized from a template from this site. Light, sensor and motors were programmed with CircuitPython on Adafruit Circuit Playground Express board. Bells were taken apart and modified to work as an octave in a circular formation with 4 motors capable of hiting left and right through creative coding. Motors were equipped with spring and ball to bounce back when hitting the bells for sound effect.
Future development will result in more color and sound effects.
