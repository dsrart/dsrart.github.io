Code Red Control, 2019

Sound and light installation/ performance
Blue LED, computer running Max/ MSP, game controller programed to manipulate sine-wave frequencies, custom mechatronic apparatus with speakers, sound card, red lasers, mirrors, electronics
Dimensions variable
Code Red Control gives the performer full control over red lasers and the sound programming to create custom on demand visualizations of the sine waves materialized as Lissajous figures by the means of a game controller with Real Time data processing by the program I wrote for Max/MSP.
